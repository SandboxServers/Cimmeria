## Part A: Scripted Mission Chains

Of 1,040 total missions, **16 have Python scripts** (plus 2 with empty `script_name` strings that reference no actual code). Scripts are auto-generated by the Atrea Script Editor and use a node-graph compilation pattern where variables are named `nNN_var_Name`.

Mission scripts live in `python/cell/missions/<Zone>/<ScriptName>.py`. Space-level scripts in `python/cell/spaces/<Zone>.py` also contain mission logic (accepting, advancing, completing missions) that is NOT in the per-mission script files.

---

### Castle_CellBlock (Praxis Starting Zone)

**Confidence**: CONFIRMED
**Zone**: Castle_CellBlock (instanced tutorial zone)
**Faction**: Praxis (System Lords)
**Total scripted missions**: 15 (8 main story + 5 hallway controllers + 2 space-script-driven)
**All missions**: Level 1

The Castle_CellBlock space script (`python/cell/spaces/Castle_CellBlock.py`) acts as the master orchestrator. It handles mission acceptance for the entire zone, including mission 622 (auto-accepted on zone load), mission 638 (accepted on entering Region2), missions 640/641 (accepted via teleport and dialog), and the hallway controller missions 682/684/686 (accepted on region entry).

#### Main Story Chain

Sequential chain, each mission must complete before the next begins:

```
622 "Arm Yourself!"
 |-> 638 "Speak to Prisoner 329"
      |-> 639 "Find Ambernol"
           |-> 640 "Hack the Rings"
                |-> 641 "Preparation"
                     |-> 680 "Escape the Cellblock"
                          |-> 681 "Mess Hall Controller"
                               |-> (682-686 Hallway Gauntlet, parallel)
                                    |-> 687 "Aftermath"
```

#### Hallway Gauntlet (Hidden Background Missions)

These 5 missions run in parallel with the main chain after the Mess Hall. All are `is_hidden=true` so the player never sees quest tracker entries. They are accepted by region triggers in the space script as the player moves through hallway areas.

```
682 "Hallway01 Controller"  ->  683 "Hallway02 Controller"
                                     (684 accepted on Region4 entry)
684 "Hallway03 Controller"  ->  685 "Hallway04 Controller"
                                     (686 accepted on Region5 entry)
686 "Hallway05 Controller"  ->  (687 accepted on Region6 entry after 686 complete)
```
